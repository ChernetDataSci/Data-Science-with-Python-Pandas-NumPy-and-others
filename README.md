# Data Science with Pandas & NumPy

This repository contains a set of hands-on Data Science projects built using **Python**, **Pandas**, **matplotlib**
**seaborn** and **NumPy** to demonstrate essential skills in data manipulation, extraction, visualization, transformation, and cleaning. Each project showcases workflows commonly used in real-world data analysis.

## 📂 Projects Included

### 📌 Project #1 — Web Scraping & Data Wrangling
- Extract data from websites or APIs
- Filter, clean, and transform data using Pandas & NumPy
- Reshape and pivot DataFrames for analysis

### 📌 Project #2 — Data Visualization & Aggregation
- Explore data using visualizations (bar charts, histograms, line charts)
- Summarize insights with `groupby()` and aggregate functions
- Create meaningful visuals to communicate results

### 📌 Project #3 — Data Cleaning Techniques
- Handle missing values and duplicates
- Use Regular Expressions (regex) for text cleaning
- Standardize and prepare datasets for analysis

## 🚀 Key Skills Demonstrated

- 📊 **Data visualization** with Matplotlib/Seaborn  
- 🧮 **NumPy** array operations and numerical computing  
- 🧹 **Data cleaning** and preprocessing  
- 🔍 **Regular expressions** for pattern matching  
- 🔄 **Merging, concatenating, reshaping, pivoting**
- 📈 **GroupBy and Aggregate functions**
- 🌐 **Web scraping or API data extraction**
